## Redis 

Radis is a in memory data structure store used as a database 
Use as a Nosql database 
Distributed Cache
Message Broker 

## Supports various data Structure 
   --| String
   --| Hashes
   --| List
   --| Sets
   --| Sorted Sets
   --| Bitmaps
   --| HyperLogLog
   --| Geo Spatial Index
